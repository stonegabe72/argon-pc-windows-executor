## About Argon PC

Argon PC is a Windows desktop tool built to run Lua scripts inside the Roblox client environment. The app is tuned for a straightforward workflow: connect to Roblox processes, launch user scripts, and do it without registration or activation keys. Its design centers on low overhead and responsive behavior, which makes it useful for quick script checks as well as more involved automation setups.

Inside the app, you will find a script hub with more than 500 community scripts grouped by game and purpose. Alongside persistent script queues and batch execution, Argon PC gives Roblox script users a practical, all-in-one workspace for Windows 10 and 11.

## FAQ

**Is Argon PC safe to use?**  
Use it at your own risk. Since the tool interacts with Roblox processes, it may conflict with Roblox's terms of service. Make sure you review the license and terms before using it.

**Will it work with future Roblox updates?**  
The auto-update engine is intended to deliver compatibility patches over time. Even so, there is no promise that it will keep working after major Roblox changes.

**How does this compare to other executors?**  
Argon PC is built around lightweight performance and a simple script hub workflow. Different executors offer different feature mixes, so it is worth comparing them against your needs.

**Can my Roblox account get banned?**  
Third-party execution tools always carry account risk. We do not guarantee protection from moderation or enforcement actions. Use carefully and preferably with alternate accounts.

**Where are my scripts stored?**  
Any scripts you add are kept locally in the `ScriptHub/LocalScripts/` folder. Nothing is uploaded or shared automatically.
